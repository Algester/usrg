  <li>Netherack</li> 
  <li>Obsidian</li>
  <li>OreGenerator</li>
  <li>NetherQuartz</li>
  <li>RedStone</li>
  <li>Sand</li>
  <li>Snow</li>
  <li>SoulSand</li>
  <li>Dust<sup>1</sup></li>
</ul>
Generators can be configured to use RF, and also limit the amount of Items generated per tick.
All recipes are unlocked when new player joins world.<br/>
